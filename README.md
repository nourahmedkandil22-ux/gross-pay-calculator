# Gross Pay Calculator

A simple Python program that asks the user for hours and rate per hour, then calculates gross pay including overtime (time-and-a-half for hours over 40).

## Example

Input:
- Hours: 45
- Rate: 10.50

Output:
- Pay 498.75

- Python
- Functions
- Conditionals
